# HKDSE-Result-Analysis

The report link is here: https://cchalvin.github.io/HKDSE-Result-Analysis/

This report mentions which factors affect Final Grades of the DSE in a statistical framework; all the data are collected from DATA.GOV, HKEAA ([https://data.gov.hk/tc-datasets/provider/hkeaa](https://data.gov.hk/tc-datasets/provider/hkeaa)) from 2017 to 2025. You can download the data via the link or directly from my updated CSV as well.

This report aims to analyze how Subject, Year, and Gender affect Final Grades. Readers can base their understanding on the results of the report to know which subjects are more favorable for getting higher grades, and vice versa. It is hoped that DSE takers and teachers will know more about the general situation of the DSE and take or suggest subjects more wisely.

The report consists of 5 main parts: trends and linear models for subjects in counting and proportion; MANOVA and linear regression on proportion Grades with Gender, Subjects, and Years; clustering Subject_Gender into 5 groups with average grade ordering; and finally, conducting MANOVA and regression analysis on clustered Subject_Gender.
